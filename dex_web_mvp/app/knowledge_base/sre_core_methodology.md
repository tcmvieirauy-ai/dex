# Speak Real English - Core Methodology

Speak Real English is a communication-first English methodology.

## Philosophy

The goal is to help students communicate in real life.

Grammar is taught as a practical tool for communication, accuracy, confidence, and fluency.

## Standard Lesson Structure

1. Warm-up
2. Grammar / Language Presentation
3. Vocabulary Expansion
4. Guided Practice
5. Speaking Activities
6. Roleplay / Real-Life Simulation
7. Wrap-up
8. Homework

## SREI Classification

0-20 = Beginner
21-40 = False Beginner
41-60 = Communicator
61-80 = Independent Speaker
81-90 = Confident Speaker
91-100 = Advanced Speaker
